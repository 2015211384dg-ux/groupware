## 1. 위험평가 방법론

### 1.1 위험 산정 공식
```
위험도 = 발생가능성(1~5) × 영향도(1~5)
```

| 점수 | 등급 | 처리 방침 |
|------|------|----------|
| 16~25 | 매우 높음 | 즉시 처리 필수 |
| 9~15  | 높음 | 3개월 내 처리 |
| 4~8   | 중간 | 6개월 내 처리 또는 수용 |
| 1~3   | 낮음 | 수용 가능 |

### 1.2 처리 옵션
- **완화(Mitigate):** 통제 수단 적용으로 위험 감소
- **수용(Accept):** 비용 대비 위험이 낮아 현 상태 유지
- **회피(Avoid):** 해당 활동/시스템 중단
- **이전(Transfer):** 보험 가입 등 외부 이전
